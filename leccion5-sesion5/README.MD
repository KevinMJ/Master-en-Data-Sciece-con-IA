# Actividad: Introducción a las Funciones con Meta AI
> El objetivo de esta actividad es iniciar tu interacción con herramientas de IA generativa. A través del chat de Meta AI, investigarás conceptos fundamentales relacionados con funciones en Python (enfoque Ciencia de Datos) y conocerás las capacidades de los modelos de lenguaje.

## Objetivo de aprendizaje
* Distinguir tipos de funciones en Python y su uso típico.
* Aplicar criterios de diseño para decidir parámetros y retornos.
* Comparar UDF (funciones con def) vs funciones lambda (legibilidad y reutilización).
* Practicar iteración de prompts: preguntar de nuevo cuando algo no quede claro.
## Herramienta a usar 
Meta AI (sitio oficial): https://www.meta.ai/

Entra al sitio, abre un chat nuevo y realiza 3 prompts distintos (uno por cada interrogante).

## Prompts (3 distintos) para usar en Meta AI

Copia y pega cada prompt tal cual (uno por pregunta). Después de cada respuesta, si algo no queda claro, haz 1–2 preguntas de seguimiento (y si puedes, captura también esa parte en pantalla).

* PROMPT 1 — Tipología de funciones en Python:
  * "Actúa como tutor de Python para Ciencia de Datos. Explícame detalladamente en qué se diferencian los distintos tipos de funciones en Python (por ejemplo: built-in, definidas por el usuario con def, lambda/anónimas, métodos (instancia/clase/estático), generadoras con yield y funciones async). Para cada tipo: (1) definición breve, (2) un ejemplo corto en Python, y (3) un caso real de cuándo usarla."
* PROMPT 2 — Arquitectura y diseño de funciones:
  * "Actúa como ingeniero/a de software. ¿Bajo qué criterios técnicos es preferible que una función definida por el usuario reciba parámetros o no? Incluye aspectos como acoplamiento, pureza, dependencias, configuración, testabilidad y reutilización. Además, ¿en qué escenarios es mejor retornar un valor frente a solo ejecutar una acción con efectos secundarios? Da ejemplos concretos en Python."
* PROMPT 3 — UDF vs. funciones lambda:
  * "Compara funciones definidas por el usuario (UDF con def) vs funciones lambda en Python. ¿En qué situaciones específicas es preferible usar def en lugar de lambda? Explica el impacto en legibilidad, depuración, documentación, pruebas y reutilización. Incluye un ejemplo con lambda y su refactor a def (y explica por qué mejora)."
