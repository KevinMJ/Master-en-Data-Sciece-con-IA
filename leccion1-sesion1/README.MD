# Actividad: Exploración de IA con Meta AI
> El objetivo de esta actividad es iniciar tu interacción con herramientas de Inteligencia Artificial Generativa. A través del chat de Meta AI investigarás conceptos fundamentales de Ciencia de Datos y conocerás capacidades de modelos de lenguaje recientes.

| Link oficial Entra aquí: https://www.meta.ai/

## Objetivo
* Interactuar con Meta AI usando prompts claros y específicos.
* Practicar seguimiento (preguntas adicionales) para aclarar términos y profundizar.
* Recolectar evidencia (capturas) y redactar una breve reflexión final.

## Instrucciones generales
1. Abre https://www.meta.ai/ e inicia un chat.
2. Realiza 3 prompts (uno por pregunta) copiando exactamente el texto de cada prompt en el chat.
3. Después de cada respuesta, haz al menos 2 preguntas de seguimiento (curiosidad y aprendizaje autodidacta).
4. Toma capturas de pantalla legibles donde se vea: tu prompt, la respuesta completa y tus seguimientos.
5. Guarda todo en una carpeta de Google Drive con permisos de lector y comparte el link como entrega.
6. Prompts requeridos
   1. Prompt 1: Python en Data Science
      1. Pega este prompt en Meta AI:“Dominio de Python en Data Science: ¿Cuál es el alcance real de Python en el    ecosistema actual de la Ciencia de Datos y qué ventajas competitivas ofrece frente a otros lenguajes para un profesional en formación?”
      2. Seguimientos sugeridos (elige al menos 2)
         1. “Dame un mapa del ecosistema Python para Data Science: librerías clave por categoría (EDA, ML, DL, MLOps) y para qué sirve cada una.”
         2. “Compárame Python vs R vs SQL vs Julia en 5 criterios prácticos (empleabilidad, velocidad, ecosistema, curva, producción).”
         3. “Si estoy empezando, ¿qué 3 mini-proyectos puedo hacer en 2 semanas para demostrar dominio básico de Python en Data Science? Incluye entregables con links públicos (GitHub/Colab).”
         4. “Mencionaste X (un término que no entiendas). Explícalo con un ejemplo sencillo y otro ejemplo técnico.”
   2. Prompt 2: La Tríada de Ciencia de Datos
      1. Pega este prompt en Meta AI: “La Tríada de la Ciencia de Datos: Explica la conexión técnica y el flujo de trabajo entre la estadística, el análisis de datos y el Machine Learning. ¿Cómo se complementan estas tres áreas en un proyecto real?”
      2. Seguimientos sugeridos (elige al menos 2)
         1. “Dame un ejemplo completo (tipo caso de negocio) y describe qué harías en cada fase: EDA, estadística inferencial, feature engineering, entrenamiento, evaluación y despliegue.”
         2. “¿Qué errores comunes pasan cuando alguien se salta la estadística y va directo a entrenar modelos?”
         3. “Explícame la diferencia entre correlación y causalidad y cómo evitar conclusiones falsas en un proyecto.”
         4. “¿Qué métricas usarías para clasificación vs regresión y por qué? Dame un mini-cheatsheet.”
   3. Prompt 3: Arquitectura de Llama 4
      1. Pega este prompt en Meta AI: “Arquitectura de Llama 4: Describe las diferencias técnicas y de uso entre los modelos Llama 4: Scout, Maverick y Behemoth. ¿Cuáles están disponibles para el público general y en qué escenarios específicos es mejor elegir uno sobre el otro?”
      2. Seguimientos sugeridos (elige al menos 2)
         1. “Resúmelo en una tabla: modelo, fortalezas, debilidades, latencia/costo relativo y casos de uso recomendados.”
         2. “Si yo quisiera construir un chatbot para soporte técnico, ¿cuál elegirías y qué trade-offs tendría?”
         3. “¿Qué significa exactamente ‘disponible para el público’? Dame opciones concretas para acceder (sitio oficial, API, repositorios).”
         4. “Define términos que mencionaste (por ejemplo: contexto, parámetros, cuantización, distilación) con analogías y ejemplos.”
7. Entrega (Google Drive)
   1. Tu entrega será un link a una carpeta de Google Drive con permisos de lector que incluya:
      1. Capturas de pantalla legibles con la interacción completa con Meta AI para cada una de las 3 preguntas (se debe ver el prompt y la respuesta).
      2. Evidencia de seguimiento: capturas donde realices preguntas adicionales basadas en las respuestas de la IA (se valora positivamente).
      3. Breve reflexión: 1 párrafo final sobre qué te sorprendió más de las capacidades de respuesta de la herramienta.
8. Cómo organizar tu carpeta (recomendado)
   * 01_prompt_python/ (capturas + nota corta con lo aprendido)
   * 02_triada_ds/ (capturas + términos nuevos)
   * 03_llama4/ (capturas + tabla/resumen)
   * reflexion_final.txt o Google Doc con el párrafo final

### Notas importantes
* Asegúrate de que tus capturas sean legibles (zoom si es necesario).
* No subas archivos “desde tu computadora” como parte del formulario: comparte enlaces públicos (Drive con permisos, GitHub, Colab, etc.).
* Demuestra curiosidad: si aparece un término desconocido, pregúntale a la IA qué significa antes de cerrar el chat.

### Cierre
Cuando termines, revisa tus capturas y asegúrate de que se vea el hilo completo por cada prompt (incluyendo seguimientos). Mantén el hábito: mientras más practiques iteración de prompts, mejor vas a guiar a la IA y más valor obtendrás de sus respuestas. ¡Sigue practicando!
