> El objetivo de este challenge es aplicar los conocimientos adquiridos dentro del tema estructuras de control de flujo, basados íntegramente en el material expuesto en la sesión y las lecturas correspondientes. 

Se espera que demuestres tu competencia en las siguientes áreas clave:

* **Toma de Decisiones**: Implementar estructuras de control para dirigir el flujo del programa según la operación a efectuar.
* **Iteración Controlada**: Utilizar ciclos para mantener la agenda activa y recorrer los registros almacenados.
* **Optimización de Procesos**: Aplicar listas y/o diccionarios para el almacenamiento y filtrado de datos dentro de la agenda.
