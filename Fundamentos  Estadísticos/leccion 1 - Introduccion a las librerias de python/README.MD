# Actividad: Exploración de Librerías con Meta AI
> El objetivo de esta actividad es iniciar tu interacción con herramientas de Inteligencia Artificial Generativa. A través del chat de Meta AI, investigarás conceptos fundamentales de Ciencia de Datos y conocerás las capacidades de los modelos de lenguaje más recientes.

Deberás ingresar al sitio oficial de Meta AI y utilizar 3 prompts distintos para resolver las siguientes interrogantes. No te limites a la primera respuesta; si algo no te queda claro, ¡pregunta de nuevo!

* Ecosistema: ¿Cuáles son las 5 librerías de Python más importantes para un Científico de Datos y cuál es el rol principal de cada una dentro de un flujo de trabajo?

* Rendimiento: Explica detalladamente por qué es más eficiente utilizar arreglos de NumPy en lugar de listas nativas de Python cuando trabajamos con millones de datos. Menciona el concepto de 'vectorización'.

* Uso técnico: Proporciona 3 ejemplos de funciones de NumPy que sean esenciales para realizar cálculos estadísticos básicos (como media o desviación estándar) y muestra cómo se aplicarían a un arreglo de una dimensión.

* Se proporciona la liga del chat de Meta AI para esta actividad: https://www.meta.ai/

## Entrega 

La entrega consistirá en compartir el texto del prompt solicitado junto con la respuesta arrojada (copiar y pegar) para poder visualizar el contenido a través de:

* Espacio de texto: Encontrarás un espacio de texto en la sección de entrega de la actividad donde deberás colocar el texto antes mencionado.

* Breve reflexión: Un párrafo final sobre qué te sorprendió más de las capacidades de respuesta de la herramienta.

Nota: Si la IA menciona un término que no conoces, intenta preguntarle qué significa antes de cerrar el chat, esto ayudará a alimentar tu curiosidad en el momento. Esto puedes incluirlo en el texto copiado de las respuestas junto con la reflexión.
