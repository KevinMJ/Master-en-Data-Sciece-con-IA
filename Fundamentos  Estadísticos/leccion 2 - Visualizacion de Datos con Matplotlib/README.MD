# Challenge: Introducción a las Librerías de Python
> El objetivo de este challenge es aplicar los conocimientos adquiridos dentro del tema introducción a las librerías de python, basados íntegramente en el material expuesto en la sesión y las lecturas correspondientes. 

Se espera que demuestres tu competencia en las siguientes áreas clave:

1. Dominio de NumPy Arreglos: 
   * Deberás crear arreglos unidimensionales y bidimensionales y utilizar funciones generadoras, y además de aplicar la técnica de cambiar la estructura de los datos.

2. Operaciones Vectoriales: 
   * Demostrarás la eficiencia de NumPy realizando cálculos matemáticos aritméticos (directamente sobre los arreglos sin utilizar ciclos for).
   
3. Visualización con Matplotlib: 
   * Graficarás dos funciones matemáticas diferentes en un mismo lienzo. Deberás personalizar la visualización y colocar leyendaf que las identifique claramente, cuidando la legibilidad.
