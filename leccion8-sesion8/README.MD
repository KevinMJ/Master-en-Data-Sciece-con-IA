Se espera que demuestres tu competencia en las siguientes áreas clave:

* Sistema CRUD: Implementar un sistema completo que permita al usuario realizar las operaciones de la gestión de datos.

* Gestión de Datos Estructurados: Organizar y manipular toda la información del sistema utilizando listas y diccionarios como estructuras de datos principales.
* Lógica y Control de Flujo: Implementar un menú interactivo y la lógica de la aplicación utilizando bucles y sentencias condicionales.
